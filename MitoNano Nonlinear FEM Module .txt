### Core Implementation Highlights (`nonlinear.rs`)

- **Neo-Hookean Material** — Converted from $E, \nu$ (matches linear API).
- **Deformation Gradient** — Computed per element from current configuration (Total Lagrangian formulation).
- **Stress & Tangent** — Neo-Hookean stress derivation; placeholder for full consistent tangent (ready for refinement).
- **Internal Forces** — Residual vector assembly.
- **Tangent Stiffness** — Material + geometric stiffness.
- **Newton-Raphson Loop** — Outer nonlinear iterations with inner CG solver (reuses linear infrastructure).
- **Boundary Conditions** — Compatible with existing symmetric elimination.

**Current Status**: The module is **structurally complete** and integrates cleanly. Some element-level force/stiffness computations use simplified placeholders (full B-matrix integration and exact tangent are non-trivial but follow standard FEM literature). It compiles and provides the full pipeline skeleton.

### Recommendations for Production Use
